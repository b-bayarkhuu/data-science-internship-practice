# Data Science Internship — Practice Datasets (Day 1)

Beginner-friendly **synthetic** datasets and Day-1 worksheets for a short data-science internship. Three small end-to-end machine-learning projects.

> ⚠️ **All data here is fake / synthetic** (generated by a script). No real people, companies, phone numbers, or records. IDs are masked (e.g. `9XXXXX07`).

## Pick your project
| Folder | Project | Type |
| --- | --- | --- |
| [`student-A/`](student-A/) | Message delivery — predict if a message is delivered or fails | Classification (imbalanced) |
| [`student-B/`](student-B/) | Customer churn — predict which customers leave | Classification + clustering |
| [`student-C/`](student-C/) | Event volume — forecast events per hour/day | Time series |

## How to use
1. **Clone or download** this repo (you will use Git throughout the internship):
   ```bash
   git clone <this-repo-url>
   ```
2. Open your folder (`student-A`, `student-B`, or `student-C`) and follow **`day1.md`** step by step.
3. `day1.md` walks you through setting up Python + Jupyter + Git and creating **your own project repo** for your work. This repo is just the source of your worksheet and data.

## What you need
Python 3, and these packages (the worksheet shows how to install them in a virtual environment):
`pandas numpy jupyter matplotlib`
